# STS8200S 基础 API 速查

主题 1：参数读取
- `StsGetParam(funcindex, testIndex, paramIndex)` 用于读取测试计划中的参数对象
- 典型用途是获取上下限、单位、测试条件、索引和结果上报句柄
- 风险点是参数名、测试索引、站点索引不一致，导致结果写错位置

主题 2：结果上报
- 常见方式是通过参数对象调用 `SetTestResult(...)`
- 结果上报必须和测试计划中的参数一一对应
- 如果只完成测量但没有上报，最终报表会缺项或数据无效

主题 3：初始化
- `HardWareCfg()` 用于全局硬件配置
- `InitBeforeTestFlow()` 用于每次测试前的初始化和安全复位
- 风险点是电源未归零、继电器状态未清、上次站点状态残留

主题 4：资源类型
- FOVI 适合强迫电压并测量电流
- QTMU 适合小电流、漏电流、静态电流
- DMM 适合高精度电压测量
- DIO 适合数字功能和逻辑电平相关测试

主题 5：工程习惯
- 先初始化，再上电，再配置测量，再执行测试，再上报结果，再恢复安全状态
- 任何保护动作都不应依赖人工记忆，而应写在测试程序里
