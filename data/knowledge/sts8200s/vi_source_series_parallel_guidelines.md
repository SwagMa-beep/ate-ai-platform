# VI source series and parallel guidelines

适用场景：
- 需要提升电压范围的高压测试
- 需要提升灌/拉电流能力的大电流测试
- LDO / DC-DC / 功率器件的边界测试
- BVDSS、Ishort、Ilimit 等需要特殊源组合的场景

允许的典型组合：
- 电压源 + 电压源串联
- 电压源 + 电流源串联
- 电流源 + 电流源并联
- 电压源 + 电流源并联

明确禁止的组合：
- 电压源 + 电压源并联
- 电流源 + 电流源串联

电压源与电压源串联：
- 适用于提高 DUT 端测试电压范围
- 上游源的 clamp 电流应大于下游源的 clamp 电流
- 一个源的输出电压必须小于另一路源的隔离电压
- 若发生 clamp，保持恒压状态的源，其输出电压必须小于发生箝流源的最大电压量程
- 对四线 Kelvin 浮动源，实际使用时 Force 与 Sense 最终都应在 DUT 引脚处短接；两路源串联叠加点也应提前完成 F/S 短接，以保证电压精度

电压源与电流源串联：
- 常见于提高可测电压范围，如功率器件击穿类测试
- 恒压源的 clamp 电流必须大于恒流源的输出电流
- 恒压源输出电压必须小于恒流源隔离电压
- Kelvin 浮动源串联时同样要保证叠加点 F/S 短接

电流源与电流源并联：
- 常见于多路 VI 通道并联提供更大电流
- 各通道的 clamp 电压都必须大于 DUT 两端电压
- 并联前要确认通道状态一致、导线电阻和路径尽量对称

电压源与电流源并联：
- 常见于 DC-DC、LDO 的 Ishort / Ilimit 等测试
- 用恒压源提供主偏置，用恒流源辅助拉载或限流
- 恒流源的箝压必须大于恒压源输出电压，避免恒压源反向向恒流源灌电流
- 不参与当前参数测试的并联通道，建议关闭输出继电器，避免倒灌损伤

绝对禁止的错误做法：
- 电压源并联：即使初始看起来电压一致，只要 DUT 内部动作让两路源互联，也可能造成源间对冲、倒灌和板卡损伤
- 电流源串联：两路恒流源串联后会失去可控性，可能导致异常电压积累和源工作点失常

工程建议：
- 先明确目标是“扩电压”“扩电流”还是“实现特殊负载条件”，再决定串/并联方式
- 组合前先检查：
  - 隔离电压
  - clamp 边界
  - 最大量程
  - Kelvin 连接点
- 对可疑组合，先用保守电压/电流档位和更紧的保护阈值做验证

常见风险：
- 忽略隔离边界，导致源间击穿或板卡损坏
- 没有关闭闲置通道，导致倒灌
- Kelvin 连接处理错误，导致测量值与 DUT 实际端电压不一致
- 只关注输出目标，不关注 clamp 方向和大小
